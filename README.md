# Supermarket Sales Data Analysis

This project analyzes supermarket sales data using Python and Google Colab.

The analysis includes:
- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- Visualizations and statistical insights
- Answers to key business questions

Tools used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

The notebook file `project.ipynb` contains the complete code and analysis.
